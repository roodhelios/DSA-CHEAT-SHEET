Variables & Types

📦
🧠 Mental Model: Sticky Notes, not labeled boxes

In Java/C++, a variable is a labeled box — the box type is fixed forever. In Python, a variable is a sticky note you attach to any object. The note can move to anything, anytime.

Dynamic typing & multiple assignment

No type declarations. Reassign to any type freely. Assign multiple variables on one line.

⚠️ Incrementing — No ++ in Python!
⚠️
n++ throws a SyntaxError. Always use n += 1.
None — Python's null

Math & Division

➗
⚠️ Division is DECIMAL by default — unlike most languages!
⚠️
Critical: -3 // 2 = -2 in Python (floors down). Java/C++ give -1 (truncates toward zero). Use int(a/b) to match other languages.
⚠️ Modulo with negatives is also different!
Full math toolkit
💡
Use float('inf') and float('-inf') as initial values when finding min/max — any real number will beat them on the first comparison.

Lists (Arrays)

📋
🧠 Mental Model: Dynamic arrays that do everything

Python lists grow automatically. They serve as arrays, stacks, or queues. The most versatile structure in Python.

Basics — create, access, modify

O(1) for append/pop at end. O(n) for insert in middle — all elements shift.

Unpacking & zip
Sorting & reversing

Strings

📝
🧠 Mental Model: Immutable list of characters

Strings work like lists — you can index and slice them. But they are frozen — you can never change a character in place. Every modification creates a brand new string.

Basics — immutability & slicing
⚠️
Every + on strings creates a new string — it's O(n). Inside loops, append to a list then join at the end.
Type conversion & ASCII
join() — the RIGHT way to build strings
💡
Read "sep".join(list) as: "join these items together with sep as the glue." The string before .join is the separator.

Hash Sets

🔍
🧠 Mental Model: A bag that instantly knows what's inside

A list checks membership by scanning every element — O(n). A set uses hashing to answer "is X in here?" in O(1) — instantly, regardless of size. Trade-off: no duplicates, no ordering.

Hash set — all operations
💡
Use sets to answer "have I seen this before?" — cycle detection, visited nodes in graphs, finding duplicates. All O(1).

Heaps (Priority Queue)

🏔️
🧠 Mental Model: A self-sorting queue — smallest always at top

After every push or pop, the heap reorganizes so the minimum value is always at index 0. You never search for the min — it's always right there.

Min heap — default in Python
Max heap — negate values (Python trick)
🧠
Negating flips the ordering: the largest number becomes the "most negative" = rises to the top of the min heap. Negate on push, negate again on pop to recover the original.
heapify — build from existing list in O(n)

⚠️ The 5 Biggest Python Gotchas
1
No ++ operator. Use n += 1 always. n++ is a SyntaxError.
2
Division is decimal. 5 / 2 = 2.5. Use // for floor division.
3
Floor division floors toward -∞. -3 // 2 = -2, not -1. Use int(-3/2) to truncate like Java/C++.
4
2D list trap. [[0]*4]*4 creates shared rows. Use [[0]*4 for _ in range(4)].
5
nonlocal for reassignment. Mutating a list works. Reassigning an int/str inside a nested function requires nonlocal.
